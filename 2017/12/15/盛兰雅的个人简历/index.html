项目经历

东大方圆（2017.09 ~ 至今）

  • 东大方圆是一款集提供校内成绩，图书馆图书，校内网络中心查询，观看电视直播，查看校园AR场景，同学之间可自由分享学习资料等于一体的软件。开发动机是由于希望能将学校多种查询系统集于一体，来方便同学们在学校的生活，以及减少同学们在食堂等查询系统的排队成本。
  • 任务
    • 由于学校各个网站并不提供所需要的数据接口，所以所有所需的数据都是由我们自行在学校各个网页上解析下来。
    • 使用了 MVC 架构完成了 ip网关，网络中心，电视墙，教务处等的编写工作。
    • 实现了应用内界面之间的跳转的其他实现方式，及从应用外部跳转到指定页面的任务。
  • 所用技术
    • 使用 MVC 架构实现主体设计
    • 使用纯代码布局，适配各个尺寸屏幕及横竖屏的切换
    • 自定义 URLRouter ，借助了原生的 openUrl方法实现了从应用外进入应用时界面的跳转，并且实现了本地推送到指定界面的推送服务。
  • 学到经验
    • 学会了对推送服务的使用，在此过程中学会了自定义URLRouter，从而实现对点击推送跳转到指定界面的需求
    • 掌握了 Git 的版本控制的使用
    • 学会了简单的自定义动画，及转场动画的实现
    • 学会了使用 Charles 对界面进行抓包，使用 Postman 实现模拟网络请求，使用 TFHpple 对 HTML 页面的数据进行提取
    • 提升了对 MVC 架构的理解，学会了优化 Model和 Controller 的设计降低了模块的耦合

EWork 2.0（2017.04 ~ 2017.06）

  • 易沃克是一款针对公司内部人员管理和任务协调的软件，包含朋友圈，签到，请示，公司各种事项等功能。

  • 任务

    • 针对 EWork1.0 中出现的种种问题进行，完全重构 EWork1.0
  • 完成登录，日报周报，公司整个大模块等的开发
  • 为整个项目添加夜间模式所用技术
  • 负责应用上架和迭代更新的提交和审核

  • 所用技术

    • 基于 Runtime 给UIView及其子类的分类添加属性实现夜间模式的开发
  • 接入第三方SDK实现短信验证和推送功能
  • 基于 MVC 框架重构了原本 EWork1.0中冗杂的Controller，实现了 Controller与 Model ， View 的较好解耦

  • 学到经验

    • 学到了 Runtime 的基本实现以及其的强大之处
    • 学习了在代码设计中的高耦合，低内聚
    • 学到了程序编写中的数据持久层的设计
    • 学到了针对于 UITableView 和 UICollectionView 的界面优化

创业汇（2017.03 ~ 2017.04）

  • 创业汇是一款利用移动互联网技术连接创业企业，政府部分，创业载体，投资机构和服务机构，实现政策信息直通、创业资源整合、创业服务共享、是一款有关互联网+创业服务平台的软件。

  • 任务

    • 进行了工具类的提取和自定义控件的编写及部分逻辑的编写
  • 利用第三方 SDK 实现第三方登录和分享
  • 软件后期的维护和迭代更新

  • 学到经验

    • 学会了解决当界面出现图片时的加载问题，解决界面滑动时的卡顿问题
    • 学会了抽取工具类和自定义控件来减少 Controller 的业务逻辑
